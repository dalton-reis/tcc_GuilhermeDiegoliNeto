PRIORIDADES DOS REQUISITOS
Crucial: Absolutamente deve estar presente na versão final
Prioritário: Preferencialmente deve estar presente na versão final
Dispensável: Importante, mas pode ser dispensado se necessário
Extra: Não necessário para o projeto base, mas interessante para extensão
